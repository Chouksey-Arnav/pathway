02 · ML Learning Path
Your ML Ladder

You're already on Phase 2. This is your full progression from where you are now to LLM engineering — the skill commanding $195K–$350K+ in 2026 market data.

Where you stand: You're doing linear algebra to prep for Harvard's ML course. That's Phase 2 of 6. Most people who want to "learn AI" skip Phases 1–3 entirely and wonder why nothing makes sense. You're doing it right. The ladder below shows you exactly where this leads and what to do in each phase.
PHASE 01
COMPLETE
Already done
Python Fundamentals
Variables, functions, loops, data structures, file I/O, OOP. You've been through this with Scratch and Python basics. The goal now is speed and fluency — writing Python without thinking about syntax.
Variables + Types Functions + Scope Lists + Dicts Loops + Conditionals Basic OOP
PHASE 02
IN PROGRESS
Right now
Linear Algebra + Calculus for ML
Vectors, matrices, matrix multiplication, dot products, eigenvectors, eigenvalues. Derivatives, chain rule, partial derivatives, gradient descent. This is the mathematical engine behind every neural network. You're building the foundation that makes ML non-mysterious. Harvard's ML course requires exactly this.
Vectors + Matrices Matrix Multiplication Dot Products Eigenvectors/values Derivatives + Chain Rule Gradient Descent
▶ Resource: 3Blue1Brown: Essence of Linear Algebra (watch alongside your prep)
PHASE 03
NEXT UP
Summer Month 2
NumPy + Pandas + Data Visualization
The Python scientific stack — how you actually work with data in ML. NumPy implements the linear algebra you just learned. Pandas handles real-world datasets. Matplotlib/Seaborn visualizes your findings. Every ML project in the world uses these. Do the Kaggle micro-courses on these — free, hands-on, and fast.
NumPy arrays + ops Pandas DataFrames Data cleaning Matplotlib basics Feature engineering
▶ Resource: Kaggle Learn (free micro-courses) — Pandas, NumPy, Data Visualization
PHASE 04
THIS SUMMER
Summer Month 3
Classical ML (scikit-learn) + Harvard Course
Linear/logistic regression, decision trees, random forests, SVMs, k-means clustering, PCA. This is where Harvard's ML course lives. Scikit-learn is the standard library for classical ML. After this phase, you can enter Kaggle competitions and build real ML pipelines. This is the unlock that transforms you from someone who knows Python to someone who builds ML systems.
Linear Regression Logistic Regression Random Forests Cross-validation Feature selection scikit-learn pipelines Kaggle Titanic
▶ Primary: Harvard ML Course + Kaggle Titanic competition

SOPHOMORE YEAR
9th → 10th grade
Deep Learning with PyTorch
Neural networks, backpropagation, CNNs (images), RNNs/LSTMs (sequences), transformers. PyTorch is the dominant framework in research and increasingly in production. The 2026 market data is clear: PyTorch production experience adds $15,000–$25,000 to job offers vs. TensorFlow-only candidates. Learn it early and learn it properly. fast.ai's course is the best free deep learning curriculum in the world.
Neural Networks PyTorch tensors CNNs Transfer learning Transformers basics GPU training
